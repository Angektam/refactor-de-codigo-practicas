# Configuración del entorno DNS entre Windows y Ubuntu

## 1️⃣ Windows como Servidor DNS, Ubuntu como Cliente
1. En Windows, ejecutar `windows_dns_script.txt` y `dns_messages.txt` con PowerShell.
2. En Ubuntu, ejecutar `ubuntu_client_script.txt` y `client_messages.txt`.

## 2️⃣ Ubuntu como Servidor DNS, Windows como Cliente
1. En Ubuntu, ejecutar `ubuntu_dns_script.txt` y `dns_messages.txt`.
2. En Windows, ejecutar `windows_client_script.txt` y `client_messages.txt`.

✅ Servidor y clientes DNS configurados.
